# Step 6: Push to remote

🧠 Explanation
git init starts a new Git repo.

git add . stages all your files.

git commit saves a snapshot.

git branch -m master main renames the branch (GitHub now prefers main).

git remote add origin ... links the repo to GitHub using your secondary account via SSH alias github-suupratik.

git push -u origin main uploads your code to that repo and sets it as default for future pushes.
